Simplified Roblox API Documentation

Welcome to the Simplified Roblox API! Below is the documentation on how to use the available endpoints.

Users

GET /api/v1/getuseridfromusername

This endpoint retrieves a Roblox user ID from a given username.

Parameters

  • username (required): The username to fetch the user ID for.

Example Request

GET https://simplifiedrobloxapi.onrender.com/api/v1/getuseridfromusername?username=example_username

Example Response

{ "user_id": 12345678 }

Error Response

{ "error": "User not found or an error occurred" }
